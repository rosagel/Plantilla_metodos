# 📚 Trabajos en Clase

Esta carpeta contiene los trabajos, ejercicios y prácticas desarrollados
durante las sesiones de **Métodos Numéricos II**.

## Estructura sugerida

```
trabajos_en_clase/
├── semana_01/
│   └── ejercicio_01.py
├── semana_02/
│   └── ejercicio_02.py
└── ...
```

## Instrucciones

- Crea una subcarpeta por semana o por tema.
- Nombra los archivos de forma descriptiva (ej. `biseccion.py`, `newton_raphson.py`).
- Incluye comentarios en tu código explicando el método utilizado.
